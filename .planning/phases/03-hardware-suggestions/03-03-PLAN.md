  - If detection fails (command exits non-zero — though cmdHwDetect never should): set `hwInfo = null`

Model picker annotation logic (modifies step 3b):
  - For each Ollama model in the picker list:
    - Call parseSizeGb inline on model.size (same logic as parseSizeGb function)
    - If hwInfo && hwInfo.vram.available && modelSizeGb !== null:
      - If modelSizeGb > hwInfo.vram.gb → append "[exceeds VRAM — will CPU-offload at 1-3 tok/s]"
      - Else → append "[fits in VRAM]"
    - If hwInfo is null OR hwInfo.vram.available is false: no annotation
  - Cloud models (sonnet, haiku, inherit): never annotated

}

    **parseSizeGb(sizeStr):**
    Parse the size string format returned by `ollama list` (e.g. "4.7 GB", "2.0 GB", "800 MB").
    Use regex: `/^([\d.]+)\s*(GB|MB)/i`
    - Match group 1 = numeric value (parseFloat)
    - Match group 2 = unit (uppercase)
    - GB → return val as-is
    - MB → return val / 1024
    - No match → return null

    **cmdHwDetect(cwd, raw):**
    Never call `error()` — always return a result object via `output()`.

    Build result:
    ```javascript
    const result = {
      vram: { available: false, mb: null, gb: null, source: null },
      ram: { available: true, bytes: os.totalmem(), gb: 0 },
    };
    result.ram.gb = parseFloat((result.ram.bytes / (1024 ** 3)).toFixed(1));
    ```

    VRAM detection — try nvidia-smi first:
    ```javascript
    try {
      const smiOut = execSync(
        'nvidia-smi --query-gpu=memory.total --format=csv,noheader,nounits',
        { encoding: 'utf-8', timeout: 5000, stdio: 'pipe' }
      );
      const lines = smiOut.trim().split('\n').map(l => parseInt(l.trim(), 10)).filter(n => !isNaN(n));
      if (lines.length > 0) {
        const mb = Math.max(...lines);   // take max across GPUs
        result.vram = { available: true, mb, gb: parseFloat((mb / 1024).toFixed(1)), source: 'nvidia-smi' };
      }
    } catch (nvErr) {
      if (!isBinaryNotFound(nvErr)) {
        // nvidia-smi present but failed (GPU error) — try PowerShell WMI fallback
        try {
          const psOut = execSync(
            'powershell -NoProfile -Command "(Get-CimInstance Win32_VideoController | Where-Object {$_.CurrentHorizontalResolution -gt 0} | Select-Object -First 1).AdapterRAM"',
            { encoding: 'utf-8', timeout: 5000, stdio: 'pipe' }
          );
          const bytes = parseInt(psOut.trim(), 10);
          if (!isNaN(bytes) && bytes > 0) {
            const mb = Math.round(bytes / (1024 * 1024));
            result.vram = { available: true, mb, gb: parseFloat((mb / 1024).toFixed(1)), source: 'wmi', note: 'limited to 4GB for GPUs with more VRAM' };
          }
        } catch (_) { /* both failed — vram.available stays false */ }
      }
      // If isBinaryNotFound: nvidia-smi not installed (non-NVIDIA machine) — try WMI fallback
      if (isBinaryNotFound(nvErr)) {
        try {
          const psOut = execSync(
            'powershell -NoProfile -Command "(Get-CimInstance Win32_VideoController | Where-Object {$_.CurrentHorizontalResolution -gt 0} | Select-Object -First 1).AdapterRAM"',
            { encoding: 'utf-8', timeout: 5000, stdio: 'pipe' }
          );
          const bytes = parseInt(psOut.trim(), 10);
          if (!isNaN(bytes) && bytes > 0) {
            const mb = Math.round(bytes / (1024 * 1024));
            result.vram = { available: true, mb, gb: parseFloat((mb / 1024).toFixed(1)), source: 'wmi', note: 'limited to 4GB for GPUs with more VRAM' };
          }
        } catch (_) { /* WMI also failed — vram.available stays false */ }
      }
    }
    ```

    NOTE: The above try/catch nesting for isBinaryNotFound can be simplified — structure it to avoid code duplication. The critical constraint is: if nvidia-smi exits with error (detected as "binary not found" OR non-zero exit) → try PowerShell WMI fallback. If WMI also fails → `vram.available: false`. Never call `error()`.
